Packer files for Windows 2016 / VMWare Fusion
=============================================

A packer files for a minimal Windows 2016 virtual machine with just enough to run docker accessible
from the host system.

Packages Included
-----------------

 * Chocolatey
 * OpenSSH
 * VMWare Guest Tools
